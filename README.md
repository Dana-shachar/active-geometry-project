# Active Geometry Project

WebGL ray marching sandbox for designing unit cells for self-locking bistable metamaterials. SDF-based, GPU-accelerated.

**Stack:** Three.js (r170), Vite, lil-gui, GLSL shaders

## Setup

```bash
cd active-geometry-sandbox
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Other commands

```bash
npm run build    # production build
npm run preview  # preview production build locally
```
